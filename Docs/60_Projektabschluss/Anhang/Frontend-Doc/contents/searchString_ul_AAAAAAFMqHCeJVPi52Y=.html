searchString

UMLParameter
Untitled :: Model :: Ingredient :: search :: searchString

Description

The string to search for. You can seperate mutliple search parameters by '|' (pipe), e.g. 'test|nüffe'. Then the result will contain ingredients that match for 'test' OR 'nüffe'.
