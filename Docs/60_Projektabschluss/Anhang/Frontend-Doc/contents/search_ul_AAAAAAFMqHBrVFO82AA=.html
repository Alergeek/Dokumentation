search

UMLOperation
Untitled :: Model :: Ingredient :: search

Description

Search for string in all ingredients. The name of the ingredient and all categories are considered.

Parameters

Direction Name Type Description
in authToken
in searchString string

The string to search for. You can seperate mutliple search parameters by '|' (pipe), e.g. 'test|nüffe'. Then the result will contain ingredients that match for 'test' OR 'nüffe'.
return (unnamed) Ingredient[0..*]
